# 🏏 IPL Win Predictor

A Machine Learning web app that predicts win probability in IPL matches.

## 🚀 Features
- Real-time prediction
- Uses match stats (score, wickets, overs)
- Includes home ground advantage
- Built using Streamlit

## 🛠 Tech Stack
- Python
- Scikit-learn
- Streamlit
